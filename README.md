# Flux Timer (倒计时)

Flux Timer 是一款专为 Mac 设计的极简倒计时工具，旨在为您提供最纯粹的专注体验。

## 功能特点

- **极简设计**: 呼吸感十足的界面，专注于当下。
- **快照条 (Snapshot Bar)**: 独特的交互方式，轻松掌握时间进度。
- **高度自定义**: 支持颜色、透明度及各种样式的个性化设置。
- **macOS 原生体验**: 完美适配 macOS 14+，支持暗黑模式。

## 安装与运行

### 依赖环境
- macOS 14.0 或更高版本
- Xcode 15.0+ (用于编译源码)

### 编译运行
1. 克隆代码仓库:
   ```bash
   git clone https://github.com/您的用户名/FluxTimer.git
   cd FluxTimer
   ```
2. 运行脚本编译并启动:
   ```bash
   ./build_and_run.sh
   ```
