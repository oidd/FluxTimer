# Flux Timer

Minimalist timing aesthetics for macOS.

Flux Timer is an immersive countdown tool designed exclusively for macOS, focusing on providing a pure and aesthetic focus experience. It turns the passage of time into a visual ceremony on your desktop.
Visit the official website to view the demo video.
https://ivean.com/fluxtimer/

## Key Features

- **Linear Drag Interaction**: Unique handle drag interaction for precise timing. A tactile way to master your seconds.
- **Flux Dynamic Island**: Elegant system-level floating notifications. Important reminders leap like light without distraction.
- **Super Shortcuts**: Hold keys and tap numbers to launch your focus instantly. The ultimate synergy between touch and efficiency.
- **Pure Minimalism**: Removing all redundancy. A transparent interface that returns efficiency to intuition.
- **Stellar Particles**: Fine particle rendering makes every setting a refined visual feast.
- **Liquid Glass Aesthetic**: Native frosted gradients and dynamic lights blending perfectly with modern macOS.

## Refined Details

- **Smart Preview**: Real-time completion preview while dragging. Stay in control of your rhythm.
- **One-Click Presets**: Quickly start Noodles, Focus, or Nap modes. Enter high-efficiency states instantly.
- **Native Keyboard Input**: Directly type numbers to set time. Flexible for all your quick timing needs.
- **42-Degree Corner Curvature**: Apple-style 42-degree continuous arcs. Every corner flows with natural comfort.
- **System Notifications**: Receive native system alerts with sound when the timer ends.
- **Extend Timer**: Customize Flux Island to quickly extend countdown duration.
- **History Retention**: Keep track of the last ended timer for quick restarts.
- **Bilingual Support**: Seamless switching between English and Simplified Chinese.

## Installation and Usage

### Prerequisites
- macOS 14.0 or higher
- Xcode 15.0+ (for building from source)

### Build and Run
1. Clone the repository:
   ```bash
   git clone https://github.com/oidd/FluxTimer.git
   cd FluxTimer
   ```
2. Run the build script:
   ```bash
   ./build_and_run.sh
   ```

# 流光倒计时

极致简约的 macOS 计时交互美学。

流光倒计时是一款专为 macOS 打造的沉浸式倒计时工具，旨在为您提供最纯粹的专注体验。让时间的流逝，也能成为一种桌面的美学。
访问官网查看演示视频。
https://ivean.com/fluxtimer/

## 核心功能

- **线性触拽，精准定义**: 独特的手柄拖拽交互，指尖滑动间精准定位每一秒。这是对时间的物理掌控感。
- **流光悬浮微岛**: 优雅的系统级悬浮通知，关键提醒如流光般轻盈跃动，不干扰，更懂你。
- **超级快捷键**: 按住快捷键并轻点数字，瞬间开启专注时刻。这是指尖与效率的终极共鸣。
- **极致简约主义**: 剔除冗余干扰，界面纯粹通透，让计时效率回归直觉。
- **星辰粒子动效**: 精细粒子交互渲染，每一次设定都是一次充满仪式感的视觉盛宴。
- **液态玻璃美学**: 原生级磨砂材质与动态光影，与现代 macOS 桌面完美浑然一体。

## 精致细节

- **智能时刻预览**: 拖拽时实时显示预计结束的具体时刻，掌控节奏，从容不迫。
- **一键场景预设**: 快速开启泡面、专注或小睡模式，点击即刻进入高效状态。
- **原生键盘交互**: 支持直接输入数字修改时间，灵活应对各种快速计时需求。
- **42度圆角曲率**: 深度适配 Apple 级 42度 连续圆弧，摒弃直角，让每一处转角都流溢着自然的温润质感。
- **系统级提示**: 倒计时结束后调用系统通知并播放提示音，确保不错过任何提醒。
- **延长计时**: 自定义流光悬浮岛，快速延长倒计时时间。
- **记录留存**: 记录最后一次结束的倒计时，支持快速重新开始。
- **双语支持**: 支持简体中文和英语双语切换。
